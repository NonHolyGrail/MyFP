Защитные шлема

Определение

Шлем — средство защиты головы воина от повреждений, наносимых, преимущественно, холодным оружием, позднее — огнестрельным. На Руси общее название индивидуальных средств защиты головы ратника назывались Наголовье или Наглавник.

Происхождение

В Первобытном мире защитные головные уборы делались из дерева, бересты, сплетённых прутьев, кожи, а также шкур животных. После того, как искусство обработки металла в древних цивилизациях достигло достаточного уровня, они получили возможность изготавливать металлические шлемы. Древнейшие шлемы, сделанные из меди и золота, найдены в царских гробницах Ура и датируются 3 тысячелетием до н. э.. Это каскообразные шлемы с низкими полусферическими куполами, большинство изготовлено из бронзы. Многие из них снабжены наушами, выполненными зацело с тульёй; на одном образце есть гребень. Такие детали свидетельствуют, что данные шлемы далеко не первые.

Однако подобные изделия были слишком дорогими, поэтому долгое время оставались недоступными большинству воинов.

Современные шлема

Существуют следующие разновидности шлемов:

Баллистические шлема

Это полиуретановая пена, изготовленная специально для поглощения энергии удара,при этом сохраняющая свои свойства при резких изменениях температуры и влажности, устойчивая к воздействию огня.

Карбоновые шлема

Предназначены для защиты от ударов тяжелыми предметами, падения с высоты.Преимуществами карбоновых шлемов можно назвать: легкий вес и высокая прочность.

Пластиковые шлема

Ударопрочные пластиковые шлемы предназначены для защиты головы вне боевых условий.Их используют на страйкбольных площадках, при проведении тактических тренировок.

Модификации современных шлемов

Если старые шлема про защиту головы, то новые про защиту головы и гаджеты. Однако, модификации требуется куда-то крепить. Обычно, существует 3 интерфейса через которые можно осуществлять улучшение своего товарища.

Шрауд

Крепится на переднюю часть шлема и служит для добавления в ваш арсенал ПНВ или экшн камеры.

Рельсы

рельсы для шлема служат для фиксации дополнительного оборудования. Используются как для замены поврежденных направляющих, так и при доработке шлемов, не имеющих в базовой комплектации боковых рельс. Направляющие планки существенно расширяют тактический функционал

Велкро(липучки)

Велко-липучки редназначены для установки на шлем дополнительного оборудования и снаряжения, а также для размещения идентификационных патчей и шевронов. Липучки могут подрезаться, в зависимости от предпочтений и модели шлема.

Существует огромное множество обвесов для улучшения практичности шлема:

1.Активные наушники, которые служат для защиты вашего слуха от громких выстрелов.

2.Чехол, который крепится на велко-липучки и нужен для маскировки шлема в различных условиях(лес,зима).

3.ПНВ(прибор ночного видения) - один из самых нажных модификаторов для шлема, который позволяет лучше ориентироваться в темноте.

4.Противовес, которые требуется для стабилизации веса на голове.

5.ИК маяк, использкющийся для обозначения "свой-чужой".

1.Визор, служащий для защиты глаз.

2.Фонарь - ...

3.Камера - человеческий видеорегистратор.

4.Подвесная система - служит для удобства ношения шлема.

5.Патчи - активная защита для шлема, позволяющая отражать снаряды танка.
